We moeten de volgorde van de groene cirkels onthouden
en herhaal de reeks door erop te klikken.
Je kunt tot 7 X 7 geraken.

In vergelijking tov de vorige examenperiode heb ik geprobeeerd om een paar elementen te verbeteren/toevoegen.
Ten eerste heb ik mijn spel overgeschakeld naar een framework7 template. Omdat ik zo bijvoorbeeld mijn alert op een mooier manier kan laten tonen. Ook heb ik dit willen doen omdat ik vind dat het zo netter gaat. En zo wou ik ook een high score toevoegen bij mijn right panel. Ten slotte heb ik mijn project willen implementeren in strict mode. Mijn High score is alleen gebruikelijk enkel op google Chrome omdat ik gebruik heb gemaakt van cokies van Google. Dus zal mijn high score alleen lokaal bruikbaar zijn. En dus kan de speler alleen in het begin zijn naam geven. Voor de Cookies heb ik alles moeten zoeken op internet, maar ik heb dit helemaal moeten aanpassen op mijn spel. Het moeilijkste was het implementeren van mijn functie ShowScore() hier maak ik mijn database. Ook had ik het moeilijk om al deze nieuwe elementen te implementeren in mijn code meer specifiek in mijn document.ready(). Jammmer genoeg is het nog niet perfect. Mijn high score kan beter maar ik heb gewoon geen tijd meer.
